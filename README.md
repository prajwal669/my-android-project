# Yakshagana Loka 🎭

An Android app to explore Yakshagana — a traditional theatre form of Karnataka.

## Problem Statement
Students and enthusiasts have no single platform to discover Yakshagana artists,
events, and performances. This app bridges that gap.

## Features
- 🔐 Firebase Authentication (Login / Signup / Forgot Password)
- 🏠 Hub — Live performance events from Firestore
- 🌟 Stars — Artist directory with detailed profiles
- 🎵 Audio — Curated Yakshagana YouTube performances
- 🤖 AI Chat — Gemini-powered Yakshagana assistant
- ℹ️ About — App info and tech stack

## Tech Stack
- Kotlin + Jetpack Compose
- Firebase Authentication & Firestore
- Google Gemini AI
- OkHttp

## Setup Instructions
1. Clone the repository
2. Open in Android Studio
3. Add your own `google-services.json` to the `app/` folder
4. Sync Gradle
5. Run on emulator or device
